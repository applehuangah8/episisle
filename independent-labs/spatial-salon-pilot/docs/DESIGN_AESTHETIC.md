# Design Aesthetic Baseline

## Visual Intent

French calm minimalism with fantasy-object metaphors.
The UI should feel like an atelier workspace, not a dashboard.

## Non-Negotiables

1. One visual focus per screen area (avoid panel overload).
2. Spatial cues over taxonomy cues.
3. Calm surfaces, soft edges, low-contrast hierarchy.
4. Interaction clarity without heavy control density.

## Composition Rules

- Top: `Project Shelf` (velvet rail, object cards)
- Center: `Salon Stage` with orrery halo, world orbits, tool charms
- Right: `Wake` card (paper, slight rotation)
- Lower: `Clutter Tray` (velvet frame) beside scoped drops + lab panels

## Color / Material Direction

- Base: ivory, mist gray, dusty violet
- Accent: muted gemstone tones by world
- Material cues: glass, velvet, matte metal, paper note

## Content Tone

- Labels are short and supportive.
- Avoid command-heavy microcopy.
- RawPour messaging must reduce pressure:
  - "Drop now, sort later."
